# ステップ1: 理想状態を描く

あるあるの問題が解決された理想状態を描いてください。

## 指示

- エピソードで発生している問題が完全に解決された世界を想像する
- デザイナーとエンジニアの関係がどうなっているか具体的に描く
- 感情面でどう変わっているかも含める
- 箇条書きで3-5個の理想の状態を列挙する
- デザイナーにとってやさしい（優しい・易しい）ものになっている必要がある

## 出力先

`thinking.md` に追記する

## 出力形式

```
【理想状態】
- デザイナーは〜〜できている
- エンジニアは〜〜している
- チーム全体として〜〜な状態になっている
```
