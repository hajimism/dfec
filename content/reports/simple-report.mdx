## シンプルな構成

このレポートは従来型の単一ファイル形式です。

カスタムコンポーネントが不要な場合は、この形式が最もシンプルです。

```tsx
// シンプルなコード例
const greet = (name: string) => {
  return `Hello, ${name}!`;
};
```

## 使い分け

- **単一ファイル** (`{slug}.mdx`) - カスタムコンポーネント不要
- **ディレクトリ形式** (`{slug}/index.mdx` + `components.tsx`) - カスタムコンポーネントを使いたい場合
